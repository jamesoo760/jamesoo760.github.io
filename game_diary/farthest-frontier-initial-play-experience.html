Appearance

前言 #

前幾天在 Youtube 看到有玩家在玩這款遊戲，一看到遊戲內容就馬上購買了，畫面風格很像 anno1800 偏寫實風，玩法卻很像終區 - 與世隔絕、冰封龐克那類的生存遊戲，只不過這次是來到物資充沛的中世紀，目前這款遊戲還在開發測試中，所以有很多 Bug 以及未完成的地方。

目前覺得有特色的地方 : #

農場 #

農場的建造方式跟其他遊戲不太一樣，玩家會先畫一塊地，然後要等很久(大概一年多)農地才會完工，你會看到他們會先除草再耕地，這我不太清楚是缺點還優點，因為玩個兩三次就不想看了，也沒多細節，不過農田邊緣也會有一些樹叢，看起來也不會一塊在那很怪，這點倒是蠻值得讚許的。

農場完工後還要選你要種植的作物，一年差不多只能種兩三個作物，冬天無法種植也無法耕地，然後以三年為一個循環，會依照種植作物讓土壤有變化，感覺還蠻複雜的。

食物 #

食物會腐壞的設定讓人又愛又恨的，別的遊戲也沒看過有這個設定，不過有時候糧食不夠加上食物腐壞確實是還蠻麻煩的[em=16]。

食材的採集點無法移動也無法種植，這點還蠻可惜的，隨著村莊的擴張，採集點就因此消失了，似乎也無法控制村民去採。

疑似 BUG 或未完成的地方 : #

  1. 墓園無法刪除，但可以在墓園地上再蓋一座墓園[em=17] !!!
  2. 城鎮中心無法移動
  3. 似乎沒有新手教學，只有指南網站和右上角的操作指示
  4. 治療所每年要花費 50 個金塊，在初期的時候有點貴...，我懷疑這要後期才蓋?
  5. 選取村民後似乎只能控制他們移動，無法命令他們做事?
  6. 今天要玩時，發現全部的存檔不見了[em=18]

結語 #

目前只玩到第二階段，要升上去就必須要玩交易所，只是目前存檔都不見了...，整體感覺來說，要慢可慢，要快可快，遊戲有難易度的調整，反應比較慢的我，全部都是選擇最簡單的難度

雖然還是有侵略者或熊來攻擊你的家園，但不會到滅村的地步，特別要注意的是，沒殺死侵略者的話，金塊會被全部搶走，原本屯到 300 多個金塊瞬間歸 0…

至於熊的話，我當初不知道還可以躲進城鎮中心讓村民用弓箭殺熊，就呆呆的看著熊一個個的追殺村民，村民都躲進家裡，導致物流停滯，因而開始出現一些問題(沒食物, 沒柴凍死...)

如果一點刺激感都不想要，則可以選和平模式，大概就會像模擬城市的玩法吧
